# Vue3仿蓝桥云

#### 介绍
Vue3+element-plus仿蓝桥云课前端，json-server模拟数据

#### 软件架构
Vue3+element-plus+vue-router+axios+pinia+echarts


#### 安装教程

1.  安装依赖 `npm install`
2.  启动json-server  `json-server --watch db.json`
3.  启动项目 `npm run dev`
